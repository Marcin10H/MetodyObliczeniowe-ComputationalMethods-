# Lab 1: Schemat Hornera | Horner's Method 🧮

## Opis po polsku

Implementacja Schematu Hornera – efektywnej metody obliczania wartości wielomianów oraz dzielenia wielomianu przez dwumian. Projekt zrealizowany w ramach przedmiotu Metody Obliczeniowe.

### 📋 Opis zadania

**Obliczenia ręczne:** Ręczne wyznaczenie wartości wielomianu $W(x)=-x^{4}+6x^{3}+4x^{2}-10x+15$ dla $x=-8$. 
**Programowanie:** Stworzenie skryptów w językach **Python** oraz **MATLAB** automatyzujących ten proces.
**Weryfikacja:** Porównanie poprawności działania algorytmów z obliczeniami teoretycznymi.

### 🧪 Wyniki

Programy poprawnie wyznaczyły wartość wielomianu (wynik: -6817), co potwierdziło stabilność numeryczną zaimplementowanej metody.

Pełny opis teoretyczny, kody źródłowe oraz zrzuty ekranu z wynikami znajdują się w załączonym sprawozdaniu PDF.

## English Description

Implementation of Horner's Scheme – an efficient method for evaluating polynomials and dividing them by a linear factor. This project was developed as part of the Numerical Methods course.

### 📋 Task Description

**Manual Calculations:** Manual evaluation of the polynomial $W(x)=-x^{4}+6x^{3}+4x^{2}-10x+15$ for $x=-8$. 
**Programming:** Development of scripts in **Python** and **MATLAB** to automate the process.
**Verification:** Comparing algorithmic results with theoretical calculations for accuracy. 

### 🧪 Results

The programs correctly calculated the polynomial value (result: -6817), confirming the numerical stability of the implemented method.
