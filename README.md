# Metody Obliczeniowe | Computational Methods 🔢

## O projekcie

Repozytorium zawiera kompletny zbiór algorytmów i sprawozdań zrealizowanych w ramach kursu **Metody Obliczeniowe** na Politechnice Świętokrzyskiej. Projekty obejmują implementacje kluczowych metod numerycznych w językach **Python** oraz **MATLAB**, koncentrując się na precyzji obliczeń oraz samodzielnym tworzeniu algorytmów bez użycia gotowych solverów (podejście "from scratch").

### 📁 Spis treści (Laboratoria):

* **Lab 1: Schemat Hornera** – Optymalizacja obliczania wartości wielomianów.
* **Lab 2: Błędy i Szeregi** – Analiza różniczki zupełnej i szeregu Maclaurina dla `arcsin(x)`.
* **Lab 3: Całkowanie Numeryczne** – Implementacja metod prostokątów, trapezów i Simpsona.
* **Lab 4: Układy Równań Liniowych** – Rozwiązywanie układów metodą eliminacji Gaussa.
* **Lab 5: Interpolacja** – Wyznaczanie funkcji przez punkty metodą Newtona i Lagrange'a.
* **Lab 6: Aproksymacja (MNK)** – Wyznaczanie trendu danych metodą najmniejszych kwadratów.
* **Lab 7: Różniczkowanie Numeryczne** – Algorytmy Taylora i Stirlinga (pochodne rzędu I).
* **Lab 8: Równania Nieliniowe** – Metody bisekcji, cięciw i stycznych (Newtona).
* **Lab 9: Równania Różniczkowe Zwyczajne** – Porównanie metody Eulera oraz RK4 (Rungego-Kutty).
* **Lab 10: Optymalizacja** – Szukanie minimum funkcji metodą złotego podziału.

## About the project

This repository contains a complete collection of algorithms and reports developed during the **Numerical Methods** course at the Kielce University of Technology. The projects include implementations of key numerical methods in **Python** and **MATLAB**, focusing on computational precision and "from scratch" algorithm development without using built-in library solvers.

### 📁 Table of Contents (Labs):

* **Lab 1: Horner's Scheme** – Optimizing polynomial evaluation.
* **Lab 2: Errors & Series** – Total differential analysis and Maclaurin series for `arcsin(x)`.
* **Lab 3: Numerical Integration** – Implementation of Rectangle, Trapezoidal, and Simpson's rules.
* **Lab 4: Linear Equations** – Solving systems using Gaussian elimination.
* **Lab 5: Interpolation** – Newton and Lagrange interpolating polynomials.
* **Lab 6: Approximation (LSM)** – Trend analysis using the Least Squares Method.
* **Lab 7: Numerical Differentiation** – Taylor and Stirling algorithms (1st order derivatives).
* **Lab 8: Nonlinear Equations** – Bisection, Secant, and Tangent (Newton's) methods.
* **Lab 9: Ordinary Differential Equations** – Comparison of Euler and RK4 (Runge-Kutta) methods.
* **Lab 10: Optimization** – Finding function minima using the Golden Section Search.
